# LinkedIn-EasyApply-PanBot

My version of LinkedIn EasyApply bot (Updated March, 2020), automating application process on LinkedIn

# Usage:
  1. Download all files
  2. Run "panbot.py"
  3. Go through GUI prompt 
  4. Have fun watching! :)
